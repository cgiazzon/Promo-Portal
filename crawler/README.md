# Módulo Crawler

Este é o módulo isolado responsável por buscar informações de outras fontes (Web Scraping / Crawling).

## Estrutura
- **/monitoramento**: Escaneamento e acompanhamento das fontes externas.
- **/extrato**: Processamento e armazenamento das informações obtidas brutas.
- **/erros**: Captura, tratamento e registro de falhas nas integrações.
- **/estatisticas**: Métricas e relatórios de funcionamento do crawler (sucessos, bloqueios, volume).
